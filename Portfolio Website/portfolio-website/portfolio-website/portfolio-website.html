This was a project created in order to help showcase inprogress and completed projects, as well as my daily learning journey.



AWS Amplify


Let's begin the process for launching the static website. Log-into your AWS account, navigate to the Service search field and type amplify. Select AWS Amplify


Getting Started


Since the Host your web app option supports static website hosting, click on Get started to begin the process.


File Uploading Options

To upload the contents of the static website, highlight the desired files, right click on the mouse and select Send to, then select the Compressed (zipped) folder option. Lastly, give the newley created folder a name.
