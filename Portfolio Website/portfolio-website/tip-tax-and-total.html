Tip, Tax, and Total

Task: Write a program that calculates the total amount of a meal purchased at a restaurant. The program should ask the user to enter the charge for the food, and then calculate the amount of a 18 percent tip and 7 percent sales tax. Display each of these amounts and the total.


Hint: You shouldn't use spaces to pad the numbers, but rather format them using format function, e.g. format(x, "10,.2f").


To Compile Code:

  • Toggle Interactive Mode On
  • Then Click on Execute
